# AR Masks Quiz

Aplicación de Realidad Aumentada desarrollada con Unity que combina detección facial, máscaras 3D y un sistema de preguntas educativas de verdadero/falso.

El proyecto está orientado a la divulgación cultural e histórica mediante interacción AR en dispositivos móviles.

---

## Características principales

- Detección facial en tiempo real con AR Foundation
- Máscaras 3D colocadas sobre el rostro del usuario
- Sistema de preguntas tipo quiz (Verdadero / Falso)
- Cambio de máscara sincronizado con cada pregunta
- Feedback inmediato con explicación
- Interfaz optimizada para dispositivos móviles (portrait)

---

## Tecnologías utilizadas

- Unity
- AR Foundation
- XR Origin
- AR Face Manager
- TextMeshPro
- GitHub para control de versiones

---

## Funcionalidad

1. La aplicación detecta el rostro del usuario
2. Se muestra una máscara 3D
3. Aparece una pregunta relacionada con la máscara
4. El usuario responde verdadero o falso
5. Se muestra si la respuesta es correcta y una explicación
6. Se puede avanzar o retroceder entre máscaras y preguntas
